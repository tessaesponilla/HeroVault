# HeroVault

An wellness app that uses psychological archetypes to help professionals manage burnout through gamified habit tracking.

## Overview

HeroVault transforms personal wellness into an adventure. Users discover their hero archetype and embark on quests designed around six Jungian-inspired character types. By completing behavioral interventions and tracking wellness metrics, users build resilience, maintain work-life boundaries, and develop sustainable healthy habits.

## Features

### Gamified Wellness System
- **HP (Health Points)** - Physical & emotional resilience
- **MP (Mana Points)** - Cognitive & psychological capacity  
- **XP & Leveling** - Progress tracking and achievement rewards
- **Quests** - Behavioral interventions across 4 categories:
  - 🏃 Physical
  - 🧠 Mental
  - 💖 Emotional
  - 📋 Organization
- **Loot System** - Symbolic items representing self-care practices
- **Hero Skills** - Unique abilities tailored to each archetype
